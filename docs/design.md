> 出典: `docs/claude design/そこねこ Design Explorations.dc.html`(2026-07-12)
> 採用案: ホーム = **1b** / 価格記録 = **1c** / 商品詳細 = **1f** /
> カテゴリ内比較 = 1h / 設定 = 1i / ログイン = 1j
> ダークモード(1k)は将来スコープ(トークンは CSS 変数化しておき、後から対応可能にする)

## トーン

クリーム地 + オレンジ/アンバーの温かい配色。丸ゴシック(M PLUS Rounded 1c)。
角丸は大きめ(カード 18px、ヘッダー下部 28px)。猫モチーフはロゴ・アイコン程度に留める。

## タイポグラフィ

- フォント: `'M PLUS Rounded 1c', system-ui, sans-serif`(Google Fonts、weight 400/500/700/800)
- 価格の数字: 800(extra bold)。底値はアクセント色 `#E8823C`、非底値は `#453A2F`
- 見出し・ラベル: 700〜800 / 補足: 500

## レイアウトパターン

- **アプリシェル**: 上部にオレンジのラウンドヘッダー(下角 28px)、下部に固定タブバー。
  タブは「ホーム / 記録(中央 FAB・+アイコン・円形 52px)/ 比較 / 設定」の 4 つ
- **ホーム(1b)**: ヘッダー内に統計サマリー 3 枚(登録商品・今週の記録・底値更新。
  白 16% の半透明カード)。本文はカテゴリ見出し(「洗剤」+「1mlあたりで比較 →」)ごとに
  白カードで商品行をグルーピング。各行 = 左に価格(底値はオレンジ・特売バッジ付き)、
  中央に商品名+「店名 · 単価」、右にシェブロン
- **価格記録(1c)**: 電卓ファースト。商品・店舗はチップで選択(その場追加可)、
  画面下半分に常時テンキー。特売トグルと内容量入力(総量ヒント付き)
- **商品詳細(1f)**: 底値ヒーロー(大きな価格+店舗+特売バッジ)→ 通常のみの底値併記
  (該当なしは「—」)→ 店舗別底値 → 履歴リスト(編集・削除)
- **特売バッジ**: 丸ピル(`#FDE7E1` 地に `#D95643` 文字、font-size 10px 程度)

## 実装方針

- カラー・角丸・影は CSS カスタムプロパティとして `src/index.css` の `:root` に定義し、
  Tailwind v4 の `@theme` で参照する。ハードコードした hex を JSX に直接書かない
- アイコンは Lucide(lucide-react)を使用
- `docs/claude design/_ds/`(Modernist デザインシステム)は**採用しない**。
  探索本体(dc.html)のクリーム+オレンジのトーンが正であり、Modernist はツール付属の別系統
